# 前言
AI 实在是太火了，做一个接地气的项目玩玩吧，准备做一个能在微信聊天使用的ChatGPT 助手，先做一个简单的功能出来

# 功能设计
# 主要功能
1.获取登录二维码
2.扫描登录
3.定时检测
4.登陆成功触发
5.查询热登录状态
6.处理消息时间
6.1处理文本回复文本
6.2处理文本回复图片

其他功能
1.按条件处理群聊消息
2.消息记录
3.下线通知
