## 语法总结

```
# 全局安装脚手架
npm install -g create-preset
```

```

# 使用 `vue3-ts-vite` 模板创建一个名为 `hello-vue3` 的项目
preset init hello-vue3 --template vue3-ts-vite
```

使用 Vite 创建项目(主流)

```
npm create vite
```
