# 前沿
最近写到一个小需求，需要用图片识别的内容 直接插入到表格，识别数量还不少，我第一时间就想到了飞书表格，在网上看到了很多飞书表格的视频教程，但是都没实际去实践，理论上是可以做这个东西的。在网上看到的是很多爆文洗稿的内容，直接复制文章的链接，调用表格的插件就就能自动解析文章的详情和数据， 然后再 写提示词来洗稿，然后再有一些自动发布的功能，看起来非常用，今天就用飞书表格的能力小试牛刀
